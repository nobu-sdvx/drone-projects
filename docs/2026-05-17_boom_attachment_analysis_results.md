# ブーム接合方式変更による影響解析 結果レポート

**作成日**: 2026-05-17
**根拠プログラム**: `2026-05-17_boom_attachment_analysis.py`(リポジトリ直下)
**実行**: `python 2026-05-17_boom_attachment_analysis.py`(`PYTHONIOENCODING=utf-8`)
**本タスクの範囲**: 数値計算のみ。計算書(§2・§3・§7)の文書修正は後続ハンドオフで実施する。

---

## 1. 背景・前提条件

2026-05-17 設計レビューで、ツインブームと主翼の接合方式の変更が決定された。

| | 旧方式 | 新方式 |
|---|---|---|
| 接合 | ブームが主翼を弦方向に貫通 | 主翼後縁から **30 mm 深埋め込み + EPS スリーブで根元補強** |
| ブーム全長 | 約 200 mm | **147 mm**(前端 158 mm 〜 後端 305 mm)|
| 狙い | — | 主翼スパーとの交差回避、流線形要件対応 |

**新方式のジオメトリ**

- ブーム前端 = 主翼後縁 188 − 埋め込み 30 = **158 mm**
- ブーム後端 = 尾翼後端 = **305 mm**、全長 = **147 mm**
- 空中露出(片持ち梁)長 = 305 − 188 = **117 mm**
- 主翼スパー(弦の 30 %、機首から 146 mm)までのクリアランス = 158 − 146 = **12 mm**(干渉なし)

**主な凍結値**: L_worst = 100 mN、σ_ult(CFRP)= 1500 MPa、τ(エポキシ)= 10 MPa、η = 0.10、
σ_eps_y = 0.4 MPa、ブーム線密度 4.87 mg/mm、EPS スリーブ OD φ5 / ID φ2.1 / L 15 mm / 密度 0.025 g/cm³。

---

## 2. 計算結果(Task 1〜7)

### Task 1 — ブーム新質量

```
m_boom = 線密度 × 全長 = 4.87 mg/mm × 147 mm = 0.7159 g/本
合計(2本) = 1.4318 g
Δm_boom = 1.4318 − 1.8(旧) = −0.3682 g(軽量化)
```

### Task 2 — EPS スリーブ質量

```
V_outer  = π × 2.50² × 15 = 294.52 mm³
V_inner  = π × 1.05² × 15 =  51.95 mm³
V_sleeve = 294.52 − 51.95 = 242.57 mm³
m_sleeve = 242.57 mm³ × 0.025e-3 g/mm³ = 0.00606 g/個
合計(2個) = 0.01213 g
```

### Task 3 — ブーム強度(空中露出区間、片持ち梁)

```
M_boom  = (L_worst/2) × ℓ_cantilever = 0.050 N × 0.117 m = 5.850 mN·m
W_s     = π·d³/32 = 0.7854 mm³(φ2 mm、不変)
σ_boom  = M / W_s = 7.45 MPa
SF_boom = σ_ult / σ = 1500 / 7.45 = 201
```

旧 SF = 118(ℓ=200 mm 仮定)→ 新 SF = **201**(ℓ=117 mm)。片持ち長が短くなり安全率は向上。**判定 ✓**(目標 SF ≥ 4)。

### Task 4 — 埋め込み部 接着強度(新規評価)

**(a) 引き抜き保持力(軸方向プルアウト)**

```
A_adhesive = π × 2.1 × 30 = 197.92 mm²
τ_eff      = τ × η = 10 MPa × 0.10 = 1.0 MPa
F_pullout  = τ_eff × A = 197.92 N
SF_pullout = F_pullout / F_axial = 197.92 / 0.05 = 3958
```

**判定 ✓**(目標 SF ≥ 4)。F_axial = 0.05 N は暫定値。

**(b) 曲げによる引き抜き(回転モード)**

```
M_embed = M_boom = 5.850 mN·m
反力作用幅 a = (2/3) × 30 = 20.0 mm
F_couple = M_embed / a = 0.2925 N
σ_eps    = F_couple /(a × d_hole) = 0.2925 /(0.020 × 0.0021) = 0.00696 MPa
SF_rotation = σ_eps_y / σ_eps = 0.4 / 0.00696 = 57
```

**判定 ✓**(目標 SF ≥ 4)。

### Task 5 — 重心位置の再計算

```
新総質量 m_total = 36.47 + (−0.3682) + 0.01213 = 36.114 g
Σmx 更新 = 5431.8 − 1.8×213 + 1.4318×231.5 + 0.01213×195.5 = 5382.23 g·mm
新重心 x_cg = 5382.23 / 36.114 = 149.04 mm
旧重心 148.94 mm からのシフト = +0.10 mm
```

ブーム CG が後方(213 → 231.5 mm)へ移り、かつ軽量化したため、重心シフトは +0.10 mm とごく僅か。

### Task 6 — 静安定性指標(wing_LE = 128.0 mm 据え置き)

```
中立点 x_NP/c = 0.497(Drela MIT 16.01、AR_h=4.111, Vh=0.55)
x_NP = 128.0 + 0.497×60 = 157.80 mm
SM   = (x_NP − x_cg)/c = (157.80 − 149.04)/60 = 14.6 %
Vh   = Sh·ℓ_h/(S·c) = 53.28×134.25/(216×60) = 0.552(x_cg に非依存、不変)
```

SM = **14.6 %**(目標 5〜15 % 内 ✓)、Vh = **0.552**(目標 0.30〜0.60 内 ✓)。

### Task 7 — wing_LE 補正の要否

```
新重心の翼弦位置 MAC% = (149.04 − 128.0)/60 = 35.06 %
wing_LE 補正量 = x_cg − 0.35·c − 128.0 = +0.035 mm
```

MAC 35.06 % は目標 35 ± 1 % 内、補正量 0.035 mm は製作公差(±1 mm)内。**→ wing_LE 補正不要**。

| 項目 | 旧値 | 新値 | 差分 | 判定 |
|---|---|---|---|---|
| ブーム質量 | 1.80 g | 1.43 g | −0.37 g | — |
| EPS スリーブ質量 | 0 | 0.012 g | +0.012 g | — |
| 機体総重量 | 36.47 g | 36.11 g | −0.36 g | 40 g 制限内 ✓ |
| ブーム SF | 118 | 201 | +83 | SF ≥ 4 ✓ |
| 埋め込み SF(プルアウト)| — | 3958 | — | SF ≥ 4 ✓ |
| 埋め込み SF(回転)| — | 57 | — | SF ≥ 4 ✓ |
| x_cg(設計点)| 148.94 mm | 149.04 mm | +0.10 mm | — |
| SM(設計点)| 14.7 % | 14.6 % | −0.1 % | 5〜15 % ✓ |
| V_h(設計点)| 0.552 | 0.552 | 0 | 0.30〜0.60 ✓ |

> **旧値の出典について**:本表の旧値 SM・V_h は 2026-05-17 の v3 整合後の現行値(§7.6.2 SM = 14.7 %、§7.6.3 V_h = 0.552)を用いた。本ハンドオフ記載の旧値(SM 14.5 %、V_h 0.567)は v3 整合前の値であり、整合により x_NP/c が 0.494 → 0.497、Sh が 54.7 → 53.28 cm² に更新済みのため本表では採用していない。詳細は `docs/analysis_results_v3.md` を参照。

**総合判定**:接合方式変更は機体を **0.36 g 軽量化**し、強度はブーム・埋め込み部とも目標 SF ≥ 4 を大きく上回る。重心・静安定・容積比はいずれも実質的に不変で、設計要件をすべて満たす。

---

## 4. wing_LE 補正の要否判定

新重心 x_cg = 149.04 mm に対し、設計点(CG @ MAC 35 %)からのずれは **+0.035 mm**。これは主翼取付公差(±1〜2 mm)を大きく下回るため、**wing_LE = 128.0 mm を補正せず据え置く**。製作後の重心実測で最終確認する(現行 §7.5 の方針どおり)。

## 5. 後続文書修正への影響リスト(提案 ―― 本タスク範囲外)

後続ハンドオフで以下の修正を提案する。実装は行わない。

**尾翼書 §3(U字版v3)**

- **§3.0 選定根拠表**「ブーム取付方式」:`主翼貫通` → `主翼後縁から 30 mm 埋め込み + EPS スリーブ補強`。
- **§3.2.4 材料・加工法**:ツインブーム自重 `約 1.8 g` → `約 1.43 g`(全長 200 → 147 mm)。尾翼アセンブリ合計 `約 2.7 g` → `約 2.3 g`(尾翼 0.9 + ブーム 1.43 + スリーブ 0.01)。
- **§3.4.4 ツインブーム強度**:片持ち長 `ℓ_boom = 200 mm` → `117 mm(空中露出長)`、M_boom `10.0 mN·m` → `5.85 mN·m`、σ `12.7 MPa` → `7.45 MPa`、SF `118` → `201`。
- **§3.5.1 接合方法**:`ブーム ↔ 主翼` の行を「主翼貫通」から「主翼後縁から 30 mm 埋め込み、EPS スリーブ(OD φ5 × L 15 mm)で根元補強、エポキシ接着」に変更。ブーム全長 200 → 147 mm、前端位置 158 mm。**埋め込み部接着強度(プルアウト SF 3958、回転 SF 57)を新規記載**。
- **§3.7 残課題**:F_axial = 0.05 N は暫定値 ―― 推力反力等の軸力の精査を追加。

**主翼書 §2**

- **§2.5 機構設計**:主翼スパー位置を `弦の 30 %(前縁から 18 mm)` と明示。新方式ではブームが主翼を貫通しないため、`§2.5 のブーム貫通穴(φ2.1 mm)`の記述を削除し、ブームは主翼後縁に突き当てて埋め込む旨に変更。
- **§2.7 残課題**:ブーム貫通穴に関する項目があれば削除。

**重量重心書 §7(U字版v3)**

- **§7.2 部品別重量内訳表**:`ツインブーム 1.8 g @ 213 mm` → `1.43 g @ 231.5 mm`。新規行 `EPS スリーブ ×2 0.012 g @ 195.5 mm` を追加。合計 36.47 → **36.11 g**、Σmx 5431.8 → **5382.2 g·mm**。
- **§7.4 重心位置**:x_cg `148.94 mm` → `149.04 mm`。
- **§7.6.2 静安定余裕**:SM(設計点)`14.7 %` → `14.6 %`(限界点も同様に再計算)。
- **§7.6.3 容積比**:V_h は ℓ_h に依存し x_cg に非依存のため **変更なし(0.552)**。
- **§7.5 主翼位置**:wing_LE 補正不要(補正量 0.035 mm)を明記。
- **§7.7 結論**:総重量・x_cg・SM の数値を更新。

**§4 胴体・接合部設計(新規章の検討)**

- 埋め込み接合 + EPS スリーブは胴体–尾翼間の接合機構であり、§4 に「ブーム埋め込み接合部」の節を新設して埋め込み深さ・スリーブ寸法・接着強度(プルアウト/回転)を集約することを提案する。
