# docs/ ―― ドローン班 概略計算書と解析資料

応用機械設計製図Ⅰ「ドローン班」課題。屋内用固定翼ドローン(総重量 40 g 以下、ツインブーム + U字型尾翼構成)の概略計算書と解析資料を収める。

---

## 🚀 他の Claude / 新規セッションへの入口(ここから読む)

1. **`../CLAUDE.md`** ―― プロジェクト全体のブリーフィング(構成・運用ルール・確定事項)
2. **`analysis_results_v3.md`** ―― 全解析値・根拠スクリプト・実行結果の整合台帳(**解析値の単一の真実**)
3. **各 `概略計算書_*.md`** ―― 提出用の最終決定事項(下表)

数値の根拠を追うなら `analysis_results_v3.md` → 該当スクリプト、設計内容を読むなら各 `概略計算書_*.md` の順。

## ⚠️ 数値を変更するときの鉄則

計算書の数値は、まず該当スクリプト(`../*.py`)を更新・再実行し、`analysis_results_v3.md` を更新してから計算書に反映する。この順序を守り、文書間・文書とコードの数値ドリフトを防ぐ(詳細は `../CLAUDE.md` §5.2)。
