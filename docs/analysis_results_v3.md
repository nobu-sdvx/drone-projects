# 解析結果サマリー(v3 整合版)

**作成日**: 2026-05-17
**目的**: 4つの概略計算書(主翼・尾翼U字版v3・プロペラ・重量重心U字版v3)に記載する数値の
**「根拠プログラム」と「実行結果」を1か所に集約**し、文書とスクリプトの数値整合を取る。

本ファイルは「ドキュメント ⇔ 解析スクリプト」の単一の真実(single source of truth)である。
計算書の数値を変更する際は、まず該当スクリプトを更新・再実行し、本ファイルを更新してから
計算書に反映すること。

**実行環境**: `venv/Scripts/python.exe`、Windows、`PYTHONIOENCODING=utf-8`
