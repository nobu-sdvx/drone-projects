## 5.0 設計対象と要件

POWERUP 4.0 の 2 個のマイクロモーターに取り付ける自作プロペラを設計する。評価軸は「設計部品が課題条件を満たすことを工学アプローチにより数値で示す」ことである。

**満たすべき要件**

| 要件 | 値 | 出典 |
|---|---|---|
| 巡航で必要な推力(1モーター) | 26.5 mN | 全機抗力 52.9 mN ÷ 2 |
| プロペラ径 | モーター軸間距離 39.85 mm 未満 | 衝突回避 |
| ブレード強度 安全率 | SF ≥ 4(目標) | 課題条件(強度設計) |
| 軸-ハブ保持 | 推力反力で脱落しない | 課題条件(機構設計) |
| 製作法 | 大学 3D プリンタ(FDM、ABS)で製作可能 | 設計方針 |

**機体側からの入力値(凍結)**

| 項目 | 値 |
|---|---|
| 機体総重量 W | 0.348 N(35.5 g) |
| 巡航速度 V | 7.5 m/s |
| 動圧 q | 34.45 Pa |
| 全機抗力(主翼+尾翼+胴体) | 52.9 mN |
| 空気密度 ρ / 動粘度 μ | 1.225 kg/m³ / 1.8×10⁻⁵ Pa·s |

## 5.1 力・モーメントの設定(ステップ①)

プロペラに作用する荷重を数値化する。

| 荷重 | 値 | 備考 |
|---|---|---|
| 巡航必要推力(1モーター) | 26.5 mN | 全機抗力 52.9 mN ÷ 2 モーター |
| 最大推力(1モーター) | 186 mN | POWERUP 4.0 公式仕様(T/W=2) |
| ブレード根遠心力(40,000 rpm) | 71.1 N | **支配荷重**。F = m·r_cg·ω² |
| ブレード根曲げ応力(空気力) | 0.79 MPa | 推力由来。遠心力応力の約 1/12 で従的 |
| 反トルク | 設計上は考慮不要 | POWERUP 内蔵 PID 制御で自動補正 |
| 翼端マッハ数(40,000 rpm) | 0.220 | M < 0.3、圧縮性影響なし |

プロペラの破壊は推力ではなく**遠心力**で起こる。0.3 g のブレードが 40,000 rpm では自身の約 2,400 倍の力(71 N ≈ 7.2 kg重)で根本を引っ張られる。したがって遠心力を支配荷重として強度設計する(§5.3)。

## 5.2 概略形状の設計(ステップ②)

### 5.2.1 主要寸法

| 項目 | 採用値 | 根拠 |
|---|---|---|
| 直径 D | 36 mm | メーカ実測 36.45 mm を近似。軸間距離 39.85 mm に対し片側クリアランス 1.93 mm を確保 |
| ピッチ P | 27 mm | メーカ実測 27.1 mm を近似 |
| ピッチ比 P/D | 0.75 | 上記より |
| ブレード数 | 2 | 左右モーターとも同仕様 ×2 個 |

### 5.2.2 ブレード平面形・ねじり角

ブレードの平面形(弦長分布)とねじり角分布は、形状・性能データが公開されている Crazyflie 45mm プロペラ(UIUC Propeller Database Volume 2、`cfnq_45p1`)の無次元形状を直径 36 mm にスケーリングして決定した。

| r/R | 弦長 c [mm] | ねじり角 β [°] |
|---|---|---|
| 0.15(根本) | 5.0 | 29.2 |
| 0.50 | 6.5 | 18.1 |
| 0.70(最大弦長) | 6.9 | 16.5 |
| 0.75(代表点) | 6.8 | 16.1 |
| 1.00(先端) | 0.8 | 11.5 |

ねじり角は根本 29° から先端 12° へ単調に減少し、各半径でブレード断面の迎角がほぼ一定になる。根本(r/R=0.15)の弦長は、強度設計(§5.3)の根太め化に合わせて 5.0 mm とした(空力形状と構造で根本弦長を一元化、独立レビュー D5 対応)。

### 5.2.3 ブレード断面翼型 ★

プロペラブレードの各断面は、半径方向に動作レイノルズ数 Re ≈ 8,000〜20,000(外翼。根元はさらに低く約 2,400)の極低 Re 域で動作する。この領域に適した翼型を選定するため、候補 4 翼型を **XFLR5(v6.62)で動作 Re 域のポーラを取得**し比較した(解析タイプ Type1、NCrit=5)。

**代表点 Re = 15,000(75%R 相当)での比較**

| 翼型 | 最大 L/D | 最大時 α | その時の CL |
|---|---|---|---|
| **薄キャンバ翼型(キャンバ 4%・厚さ 2%)** | **23.4** | 5.5° | 0.90 |
| AG12(低 Re 翼型) | 17.4 | 5.0° | 0.74 |
| E205 | 8.4 | 7.5° | 0.69 |
| S1223 | 8.2 | 4.5° | 0.86 |

数値解析の結果、**薄く低キャンバの翼型が他候補を大きく上回った**。物理的には、極低 Re 域では翼面上の境界層が剥離しやすく、厚く高キャンバな翼型(S1223 等)では大規模な層流剥離により抗力が急増する。一方、薄い翼型は剥離点が前縁付近に固定され抗力増加が抑えられる。

本設計ではブレード断面翼型に**薄キャンバ翼型を採用する**。その動作点で CL ≈ 0.9 を確保し、推力に必要な揚力を発生できる。推力解析(§5.5)で用いる QPROP の翼型係数は、この翼型の XFLR5 ポーラ(Re=15,000)からフィットした。

> 📌 注記(D1 — 厚さの実現性):解析に用いた厚さ 2% は空力上の理想値である。弦長 4〜7 mm のブレードで厚さ 2%(= 0.1 mm 前後)は FDM 3D プリンタの最小肉厚を下回り、そのままでは製作できない。実機ブレードの断面厚さは印刷可能な最小値(概ね 0.4〜0.8 mm)に従い、外翼で 6〜12% 厚に相当する。本概略設計では「**極低 Re 域では薄い断面が有利**」という相対的結論を設計根拠とし、印刷可能な厚さでの翼型再評価は Phase 3 詳細設計で行う。

### 5.2.4 材料

材料は **ABS(3D プリント)** とする。大学プリンタの標準材料でコスト負担がなく、靭性が高い。

| 材料定数 | 値 | 出典 |
|---|---|---|
| 密度 ρ_ABS | 1.04 g/cm³ | 一般文献値 |
| 降伏応力 σ_y | 40 MPa | 一般文献値(バルク材) |
| 弾性率 E | 2.0 GPa | 一般文献値 |

> 📌 注記(D6):σ_y = 40 MPa は射出成形バルク材の値である。FDM 3D プリント品は積層界面(Z 方向)強度がバルクの 50〜70% 程度に低下する。強度評価(§5.3)が成立するためには、ブレードを**積層方向がスパン方向に走る向き**で印刷し、遠心引張を積層界面に沿わせる必要がある。印刷向きは §5.3.4 で指定する。

## 5.3 強度設計(ステップ③)

### 5.3.1 遠心力とブレード根応力

支配荷重は遠心力である。ブレード重心半径を保守的に 0.75R(= 13.5 mm)とした(テーパ形状により実際の重心はこれより内側にあり、遠心力を安全側に見積もっている)。ブレード質量は天秤実測前の暫定値 0.3 g を用いる。

根本断面は弦長 5.0 mm × 厚さ 1.5 mm(= 7.5 mm²)とする。これは Crazyflie 原形(根本弦長 5.5 mm)に近づける「根太め化」による(設計書 v0.3 §5.2)。

```
40,000 rpm(モーター公式最大、強度評価の最悪ケース):
  ω = 2π × 40,000 / 60 = 4,189 rad/s
  遠心力     F  = m·r_cg·ω² = 0.3×10⁻³ × 0.0135 × 4,189² = 71.1 N
  引張応力   σ_T = F / A    = 71.1 / 7.5×10⁻⁶          = 9.48 MPa
  曲げ応力   σ_B                                        = 0.79 MPa
  合成応力   σ  = σ_T + σ_B                             = 10.27 MPa
  安全率     SF = σ_y / σ  = 40 / 10.27                 = 3.9
```

曲げ応力 σ_B は、最大推力 186 mN を有効長 16 mm に等分布荷重と仮定して算出した(保守的にモーター推力全量を 1 枚のブレードに負荷)。遠心力による引張応力の約 1/12 で従的である。

### 5.3.2 動作回転数による安全率

| 動作点 | 回転数 | 合成応力 | 安全率 SF | 判定 |
|---|---|---|---|---|
| 最悪ケース(モーター公式最大) | 40,000 rpm | 10.27 MPa | **3.9** | △ 目標 4 をわずかに下回る |
| 巡航(QPROP 確定) | 約 25,100 rpm | 4.53 MPa | **8.8** | ✓ |

最悪ケース 40,000 rpm は「全開・静止」という実飛行ではほぼ生じない過渡的な極限条件であり、そこでも SF = 3.9 を確保する。実際の動作域(巡航 約 25,100 rpm 以下)では SF ≥ 8.8 と目標 4 を大きく上回る。

> 📌 注記(D3):設計書 v0.3 §5.4 は実動作の安全率を SF = 8.9(27K)/ 12.9(22K)としていたが、これは手計算の算術誤差で、正しい値は上表のとおり。スクリプト検算値を採用している。

### 5.3.3 感度解析(ブレード質量の不確実性)

ブレード質量は天秤実測前のため、0.2〜0.5 g の範囲で安全率の感度を確認した。

| ブレード質量 | SF @ 40,000 rpm(最悪) | SF @ 約 25,100 rpm(巡航) |
|---|---|---|
| 0.2 g | 5.6 | 12.2 |
| 0.3 g(暫定値) | 3.9 | 8.8 |
| 0.4 g | 3.0 | 6.9 |
| 0.5 g | 2.4 | 5.7 |

実動作域(巡航 約 25,100 rpm)では、最悪想定の 0.5 g でも SF = 5.7 を維持する。一方、過渡的な 40,000 rpm では質量が 0.32 g を超えると SF が 4 を下回る。ブレード質量の天秤実測(Phase 2.5)で確定させる。

### 5.3.4 印刷方向の指定

§5.2.4 の注記のとおり、FDM 品の積層界面強度はバルク値より低い。ブレードは**積層線がスパン(半径)方向に走る向き**で印刷し、遠心引張荷重を積層界面に沿う方向に受けさせる。これにより上記の安全率評価(バルク σ_y 基準)が成立する。寝かせ印刷(積層線が遠心力を横切る向き)は禁止とする。

## 5.4 機構設計(ステップ④)

プロペラハブをモーター軸に固定する。推力反力(最大 186 mN)がハブを軸から抜く方向に作用するため、これに耐える保持力を確保する。

| 項目 | 採用値 | 根拠 |
|---|---|---|
| モーター軸径 | φ1.00 mm | 実測 |
| ハブ穴径 | φ0.95 mm | 締めしろ −0.05 mm、圧入で初期摩擦保持 |
| ハブ長さ | 8 mm | 接着面積の確保 |
| ハブ外径 | 5 mm | 接着剤の回り込み余裕 |
| 固定方式 | 圧入 + エポキシ接着 | 二重保険 |

**保持力の計算**

```
接着面積   A      = π × d軸 × Lハブ = π × 1.0 mm × 8.0 mm = 25.1 mm²
保持力     F_hold = τ × A × η      = 10 MPa × 25.1 mm² × 0.10 = 25.1 N
安全率     SF     = F_hold / 推力反力 = 25.1 / 0.186 = 135
```

エポキシのせん断強度 τ = 10 MPa(保守値)、有効率 η = 10%(製作精度・接着ムラを織り込む保守係数)としても、保持力は推力反力の 135 倍であり、ハブ脱落のリスクは十分小さい。

[図 §5-5] `hub_attachment_detail.png` … 軸-ハブ嵌合の詳細図

**組立工程の品質保証**(計算上の安全率を実機で担保する)
1. ハブ穴を φ0.95 mm で印刷後、必要なら 0.1 mm リーマで仕上げ
2. 接着前に軸とハブ穴をイソプロピルアルコールで脱脂
3. エポキシ塗布後、1 分間圧入を保持
4. 24 時間養生(エポキシ完全硬化)
5. 試運転前に手で 2〜3 N の引張強度確認

## 5.5 推力性能の確認

自作プロペラの推力を、QPROP(MIT で開発されたブレード要素運動量法のプロペラ性能解析プログラム)で評価した。QPROP の翼型係数は §5.2.3 の薄キャンバ翼型の XFLR5 ポーラ(Re=15,000)からフィットしている。併せて、独立に実装したブレード要素運動量法による自作解析、および UIUC 実測データと照合した。

**(a) 最大能力** — 回転数 40,000 rpm・静止状態での推力は 188.6 mN と算出され、POWERUP 4.0 の公式最大推力 186 mN とよく一致する。

**(b) 巡航必要推力の到達** — 巡航(V = 7.5 m/s)で 1 モーターあたり必要な 26.5 mN は、約 25,100 rpm で発生する(QPROP)。これはモーター上限 40,000 rpm の約 63% であり、十分な回転数余裕がある。この動作点でのプロペラ効率は η ≈ 0.53 と算出され、プロペラは効率ピーク付近で巡航する。

> 📌 注記(D2 — 動作 RPM の修正):設計書 v0.3 §3.5 はピッチ幾何からの逆算で巡航動作 RPM を 22,100 rpm と推定したが、スリップ率の仮定(0.25)が小さすぎた。推力釣り合いから求まる実際の巡航 RPM は約 25,100 rpm である(実効スリップ ≈ 0.35)。22,100 rpm では推力は 14 mN(必要量の 53%)にとどまる。

**(c) 交差検証** — UIUC Propeller Database の Crazyflie 45mm 実測静推力データと照合した。同一回転数 20,000 rpm・直径 36 mm 換算で、自作プロペラの QPROP 解析値 46.5 mN に対し Crazyflie 実測換算値 37.6 mN、乖離 23.5%。極低 Re 域の解析不確実性(±20〜35%)の範囲内で実測トレンドと整合する。QPROP 値は実測換算値より約 24% 高め(非保守側)であり、推力余裕はこの不確実性を込みで評価する。

独立実装の自作ブレード要素運動量解析とは、設計点(巡航 25,122 rpm)で推力が 3.6% 一致した。ただし両者は同じ翼型ポーラ・同じブレード形状を入力とするため、これは解析手法の独立検証ではなく実装整合の確認である。手法の独立検証は (c) の UIUC 実測照合による。

> 📌 注記:設計書 v0.3 §4.7 の推力検算には算術誤差があった。Crazyflie 45mm @ 19,860 rpm の実測推力を 0.016 N としていたが、推力係数 CT = 0.165 から正しく計算すると **T = CT·ρ·n²·D⁴ = 0.091 N**(9.2 g重)である。これに基づく「推力マージンが薄い」との結論は誤りで、訂正後は推力に十分な余裕がある。

## 5.6 結論 — 要件適合のまとめ

| 要件 | 判定 | 根拠 |
|---|---|---|
| 巡航必要推力 26.5 mN を発生 | ✓ | 約 25,100 rpm で到達、モーター上限 40,000 rpm の 63% |
| プロペラ径 < 軸間距離 39.85 mm | ✓ | D = 36 mm、片側クリアランス 1.93 mm |
| ブレード強度 SF ≥ 4 | △ | 実動作域(≤ 約 25,100 rpm)で SF ≥ 8.8。過渡的最悪 40,000 rpm で SF 3.9 ― ブレード質量と印刷方向に依存。天秤実測・印刷向き確定で詰める |
| 軸-ハブ脱落防止 | ✓ | 嵌合 SF = 135 |
| 3D プリントで製作可能 | △ | 平面形・寸法・ハブは製作可能。翼型厚さ 2% は空力上の理想値で、実機は印刷可能厚に従う |

推力・寸法・機構の 3 要件は満たす。強度は実動作域で目標を大きく上回るが、過渡的最悪ケース(40,000 rpm)はブレード質量と印刷方向に依存して目標 4 を下回りうるため、天秤実測と印刷方向の確定で詰める。翼型は数値解析により薄翼が有利と確認したが、厚さ 2% は理想値であり実機は印刷可能厚で再評価する。

| D4 | 翼型解析を XFLR5 に一本化 | **解決** ― スクリーニング・QPROP 係数フィット・自作 BEM をすべて XFLR5 ポーラ基準に統一 |
| D5 | 根本弦長が空力と構造で不一致 | **解決** ― ブレード根本弦長を `C_ROOT`(5.0 mm)に一元化 |
| D6 | FDM ABS の積層界面強度低下 | **残** ― 印刷方向を §5.3.4 で指定。Phase 3 でデータシート値を再確認 |

天秤によるブレード質量の実測(Phase 2.5、§5.3.3 の感度解析の確定)も残課題である。
D2 について、`propeller_config.py` の `RPM_CRUISE` を 25,122 rpm に更新し、構造解析スクリプトの三点評価ラベルも「25K巡航」に更新した。最大速度側の `RPM_MAX_OP`(V=9 m/s)は同じくスリップ逆算の推定値であり、Phase 3 で再評価する。
