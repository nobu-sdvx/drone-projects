## 1. Phase 2 の独立レビュー

解析スクリプト 4 本(`propeller_design_analysis.py` / `propeller_structural_analysis.py` / `uiuc_propeller_comparison.py` / `propeller_airfoil_screening.py`)、QPROP 連携、XFLR5 ポーラ・UIUC データを精査した。

### レビュー依頼書 §4 の 2 大発見 ― 独立検算で確認

- **§4.7 推力の算術誤差**: Crazyflie 45mm @ 19,860 rpm の実測推力。設計書 v0.3 は 0.016 N としていたが、推力係数 CT=0.165 から T = CT·ρ·n²·D⁴ を正しく計算すると **0.091 N**(9.2 g重)。約 5.5 倍の誤差を確認した。物理的にも、総重量 ≈27g の Crazyflie が 1 発 1.65g重では浮上不能で、9.2g重級が妥当。
- **翼型**: 極低 Re では薄翼が剥離しにくく有利。XFLR5 実ポーラで裏付けを確認した。

### 新たに発見した不整合(D1〜D6)

| # | 内容 | 重大度 |
|---|---|---|
| D1 | 翼型厚さ 2% は弦長 4〜7mm では FDM 3D プリント実現不可 | 高 |
| D2 | 巡航 RPM 22,100 は誤り(スリップ率の過小評価) | 高 |
| D3 | 設計書 §5.4 の SF 手計算(8.9 / 12.9)も算術誤差 | 中〜高 |
| D4 | 翼型スクリーニングが NeuralFoil 系 / XFLR5 系で 2 系統並存 | 中〜高 |
| D5 | 根本弦長が空力(4.4mm)と構造(5.0mm)で不一致 | 中 |
| D6 | FDM ABS の層間異方性(σ_y=40MPa はバルク材値) | 中〜高 |

## 2. ブレード断面翼型の確定

XFLR5(v6.62)で候補 4 翼型の動作 Re 域ポーラを取得し、代表 Re=15,000 で比較した。

| 翼型 | 最大 L/D |
|---|---|
| **薄キャンバ翼型(camber4_thick2、キャンバ4%・厚さ2%)** | **23.4** |
| AG12 | 17.4 |
| E205 | 8.4 |
| S1223 | 8.2 |

**採用: 薄キャンバ翼型(camber4_thick2)。** 極低 Re での薄翼有利は教科書的事実であり、審査会で工学的に説明できる。ただし厚さ 2% は空力上の理想値で、実機ブレードは印刷可能厚に従う(D1、Phase 3 で再評価)。

---

## 3. XFLR5 一本化(D4 解消)

計算書の解答根拠を XFLR5 に統一した(理由: CLAUDE.md §4.5 で提出資料は AI ラベルを外す方針 / §2 主翼章との一貫性 / XFLR5 ポーラは実データで再現可能)。

- `load_airfoil_polars()` を NeuralFoil 呼び出しから XFLR5 ポーラ `.txt` 読込へ全面置換。
- これによりスクリーニング・QPROP 翼型係数フィット・自作 BEM がすべて XFLR5 ポーラ起点になった。
- QPROP の `.prop` を camber4_thick2 の XFLR5 ポーラ(Re=15,000)から再フィット・再生成。

---

## 4. D5 解消 ― 根本弦長の一元化

`load_blade_geometry_from_crazyflie()` がブレード根本に `C_ROOT`(5.0mm)を使うよう変更。空力形状と構造で根本弦長が一致した。Crazyflie 参照データ(`BLADE_GEOMETRY`)は無加工のまま保持。

---

## 5. D2 ― 巡航 RPM の確定

QPROP の推力釣り合いで巡航 RPM を確定した:**V=7.5 m/s で必要推力 26.5 mN を発生する RPM = 25,122 rpm**(モーター上限 40,000 rpm の 63%)。設計書 v0.3 の 22,100 rpm はスリップ率仮定(0.25)の過小評価によるもので、実効スリップは ≈0.35。`propeller_config.py` の `RPM_CRUISE` を 22,100 → 25,122 に更新した(承認のうえ実施)。

付随して、構造スクリプトのサマリー部にラベル変更で生じた `KeyError` を修正(`RPM_EVALUATION` 駆動の動的生成に変更)。
